# 板块动量扫描仪 (Sector Momentum Scanner)

**用于 Hermes Agent 的 A股板块轮动信号技能**

捕捉主流板块 ETF 轮动信号，识别启动/加速/衰竭/机会流失阶段，辅助判断持仓板块轮动节奏。信号输出为客观状态描述，不构成任何买卖指令。

---

## 合规声明

本技能产出的所有信号均为**板块轮动状态的客观描述**，仅供辅助判断板块轮动节奏与持仓健康度，**不构成任何投资建议**，不能作为买卖决策的依据。

**不适用的场景**：选基推荐、择时决策、单一指标驱动决策、作为唯一的风控依据。

## 信号类型

| 标签 | 含义 |
|------|------|
| 🔴 启动观察 | MACD金叉 + RSI>50 + 成交量放大，等待价格确认 |
| 🟡 趋势增强 | 启动后出现20日新高，或RSI上穿60 |
| 🟢 动量加速 | RSI 65-75 + 赔率>1.5 + 均线多头 |
| ⚪ 高位整固 | RSI>70 持续，回撤风险上升 |
| 🟤 过热回落 | RSI从70+快速回落至40-55 + 成交量萎缩 |
| 🟤 机会流失 | 均线破位 + MACD死叉 + RSI<40（同日满足） |
| 🔵 相对走弱 | RSI<40 + MACD死叉 + 均线空头 |
| ❌ 数据获取失败 | 接口异常，不生成任何信号 |

> ⚠️ RSI > 70 不等于高位反转依据。趋势 ETF 可长期维持 RSI 70+，真正的高位反转信号是「过热回落」或「机会流失」三条件同时满足。

---

## 轮动评分框架

综合得分 = **相对强弱分(±40)** + **趋势强度分(±30)** + **量能验证分(±20)** + **基准偏离分(±10)**

以沪深300（000300）为绝对基准，计算各板块5日/10日/20日超额收益。

## 触发场景

| 场景 | 触发条件 | 输出侧重点 |
|------|---------|-----------|
| 日报嵌入 | 周一至周五 14:30 | 持仓关联、相对强弱、排名变化 |
| 用户主动询问 | 问"今天有哪些板块在动" | 全量排名、今日信号、重点关注 |
| 持仓检查 | 问"我的 XX 板块怎么样了" | 单一板块深度信号 + 趋势阶段判定 |
